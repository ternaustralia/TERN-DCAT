Namespaces

TERN DCAT makes extensive use of terms from other vocabularies, in particular Dublin Core [[?DCTERMS]]. TERN DCAT defines some classes and properties of its own in order to cover some metadata sections from ISO 19115-3 not already addressed by DCAT [[?vocab-dcat-3]] or DCAT-AP [[?DCAT-AP]].

Literal

Definition
A literal value such as a string or integer; Literals may be typed, e.g. as a date according to xsd:date. Literals that contain human-readable text have an optional language tag as defined by BCP 47 [rfc5646].

This specification does not impose any additional requirements to properties for this entity.
